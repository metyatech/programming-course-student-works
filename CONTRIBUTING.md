# Contributing

Thank you for your interest in contributing to `programming-course-student-works`.

## Development setup

```bash
git clone https://github.com/metyatech/programming-course-student-works.git
cd programming-course-student-works
```

## Submitting changes

1. Fork the repository and create a feature branch.
2. Open a pull request with a clear description of the change.

## Scope

This repository stores **static student works** for the Programming course, intentionally separated from `programming-course-docs` to keep the course content repo small. Please keep PRs scoped to student work files.
